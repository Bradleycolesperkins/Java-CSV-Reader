Detailed Description

A class that creates a single Bubble Chart or two Bubble Chart visualizations from a 2D Array of data \ see Graph.java.

Date
30/04/13
See Also
This class is responsible for making the Bubble chart It was made using JFreeChart and following examples on how to set up the Bubble chart It uses the Graph.java to return a JFreeChart
