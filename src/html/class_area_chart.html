Detailed Description

A class that creates a single Area Chart or two Area Charts visualizations from a 2D Array of data \ see Graph.java.

Date
30/04/13
See Also
This class is responsible for making the Area chart It was made using JFreeChart and following examples on how to set up the Area chart It uses the Graph.java to return a JFreeChart
