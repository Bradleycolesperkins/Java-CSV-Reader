Detailed Description

A class that creates a slideshow of a single Chart or two Chart visualizations \ see Graph.java.

Date
30/04/13 This class is responsible for making the slideshow It was made using JFreeChart and following examples on how to set up the charts It uses the Graph.java to return a JFreeChart
