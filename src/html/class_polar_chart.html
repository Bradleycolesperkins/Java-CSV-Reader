Detailed Description

A class that creates a single Polar Chart or two Polar Chart visualisations from 2D Arrays of data \ see Graph.java.

Date
14/04/13
See Also
This class is responsible for making the Polar chart It was made using JFreeChart and following examples on how to set up the Polar chart It uses the Graph.java to return a JFreeChart
